# Moderator Bot — полное руководство

## Запуск
```bash
npm install
node bot.js
```

## Права бота в группе (обязательно все 4)
- Удалять сообщения
- Банить участников
- Ограничивать участников (мут)
- Добавлять администраторов ← нужно для снятия с должности

---

## Что отслеживается автоматически

| Правило | Что ловит | Наказание |
|---------|-----------|-----------|
| 1+9 | Оскорбления личности | 1 оскорбление = 1 варн. 3 варна = кик. 5 = бан навсегда |
| 2 | 5+ стикеров/гифок/эмодзи за 30 сек | Удаление + мут 30 мин |
| 3 | 18+ без спойлера | Варн + удаление |
| 4 | 3+ входов/выходов за 10 мин | Бан 24 часа |
| 5 | Конфликт (6+ агрессивных сообщ. за 60 сек от 2+ людей) | Варн всем участникам |
| 6 | Расизм / оскорбление религий / политика / оскорбление президентов | Бан навсегда |
| 7 | Реклама t.me/ | Варн + удаление |
| 7 | Ссылки (1-е нарушение) | Варн + мут 10 мин |
| 7 | Ссылки (2-е нарушение) | Бан навсегда |
| 8 | Расчленёнка без спойлера | Варн + удаление |
| 9+10 | Администратор оскорбляет (1-й раз) | Предупреждение |
| 9+10 | Администратор оскорбляет (2-й раз) | Снятие с должности |
| 11 | Капс (25+ букв, 70%+) | Варн + удаление |
| — | Флуд (5+ сообщений за 5 сек) | Мут 10 мин |

**Важно:** обычные маты ("сука", "блять") — НЕ считаются. Только оскорбления личности.

## Команды администраторов

### Снятие наказаний (ответом на сообщение или @username)
```
-варн   или   -warn    — снять варн
-мут    или   -mute    — снять мут
-бан    или   -ban     — снять бан
```

### Выдача наказаний
```
/warn [причина]
/mute [время] [причина]   — 10m, 2h, 1d, 1w
/ban [время] [причина]    — без времени = навсегда
/kick [причина]
/unban
/unmute
```

### Разрешить ссылку
Ответь `!можно` на сообщение с ссылкой — ссылка будет разрешена.

### Статистика
```
/warns          — все варны
/mutes          — все муты
/bans           — все баны
/info @user     — подробная информация
/history @user  — история нарушений
```

### Общее
```
/rules  — правила
/help   — помощь
```
